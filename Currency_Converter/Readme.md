# 💱 Java Swing Currency Converter

A simple desktop **currency converter** built with Java Swing. It allows conversion between multiple currencies using fixed exchange rates.

## 🖥️ Features

- GUI-based using `JFrame`, `JComboBox`, and `JTextField`
- Supports conversion between:
  - USD 🇺🇸
  - INR 🇮🇳
  - EUR 🇪🇺
  - GBP 🇬🇧
  - JPY 🇯🇵
- Fixed exchange rates (base: USD)
- Simple and beginner-friendly interface
