# ✈️ Airplane Ticket Booking System - Java Swing

A simple desktop-based airplane ticket booking system built using Java Swing. This project demonstrates core Java GUI concepts such as combo boxes, layout managers, action listeners, and basic input validation.

---

## 📌 Features

- ✅ Passenger name input with validation  
- 🧭 Dropdowns for selecting source and destination  
- 💺 Seat selection from available options  
- 🎫 Class selection: Economy, Business, First Class  
- 💰 Dynamic price calculation based on selected class  
- ⚠️ Input validation (e.g., same source and destination)  
- 💳 Payment confirmation via dialog  
- 📋 Booking summary with passenger and travel details
